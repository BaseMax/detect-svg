# detect-svg

Detect a string or buffer is SVG or not (javascript, nodejs)

### Installing

soon:

```
npm i detect-svg
```

### Using detect-svg
```js
console.log( detectSVG('<svg xmlns="http://www.w3.org/2000/svg"><path fill="#00CD9F"/></svg>') )
```
